Correlations in R

Use IRIS dataset available in R

d <- iris
head(d)

plot the data in iris to visualize relations between all attributes of the dataset

plot(d)

Correlation only works on numerical attributes

d <- iris[,-5]
head(d)
cor(d)
##              Sepal.Length Sepal.Width Petal.Length Petal.Width
## Sepal.Length    1.0000000  -0.1175698    0.8717538   0.8179411
## Sepal.Width    -0.1175698   1.0000000   -0.4284401  -0.3661259
## Petal.Length    0.8717538  -0.4284401    1.0000000   0.9628654
## Petal.Width     0.8179411  -0.3661259    0.9628654   1.0000000

Correation value range from -1 to +1, with -1 denoting inverse correlation, +1 denoting linear correlation and 0 no correlation

‘cor’ function can be used with different algorithms like kendall and spearman

cor(d,method="kendall")

plotting the correlation matrix

cp <- cor(d)
library(corrplot)
## Warning: package 'corrplot' was built under R version 3.6.1
## corrplot 0.84 loaded
corrplot(cp)

method arg should be one of “circle”, “square”, “ellipse”, “number”, “shade”, “color”, “pie”

corrplot(cp,method="pie")

corrplot(cp,method="color")

corrplot(cp,method="number")

corrplot(cp,method="shade")

type arg should be one of “full”, “lower”, “upper”

corrplot(cp,type="upper")
